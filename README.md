# inee-planea
Funciones para manipular las tablas de datos Planea y Excale, pruebas de evaluación a gran escala del aprendizaje diseñadas por el INEE de México.

## 09_errores_replicados
Estimación de media poblacional con su error estándar asociado e intervalo de confianza (95%).

## 10_porcentajes_errores_replicados
Estimación de porcentaje con su error estándar asociado e intervalos de confianza (95%).

## 12_multiples_grupos
Estimación de media poblacional para multiples variables simultaneamente. Incompleta.

## 13_valores_plausibles
Estimación de media población para variables con puntuaciones plausibles, con su error estándar asociado e intervalo de confianza (95%).
